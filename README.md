Comic Viewer
============
A small script to serve a folder as an comic viewer webpage.

Dependencies
------------
* node.js
* opener

Todos
-----
* In firefox with overflow hidden, the search next won't jump to correct position.
* Smart page up/page down.
* Smart left/right.
* Filter.

/view?path=PATH
/src?path=PATH
